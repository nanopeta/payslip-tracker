# 給与明細・源泉徴収票ダッシュボード

給与明細（月次）と源泉徴収票（年次）をPDFからアップロードして管理・可視化するWebアプリ。

**URL**: https://nanopeta.github.io/payslip-tracker/

## 特徴

- **完全ローカル処理**: PDFの解析・データ保存はすべてブラウザ内のみ。外部サーバーへの送信なし
- **PDF自動解析**: 給与明細・源泉徴収票を自動判別してデータ抽出
- **月次グラフ**: 差引支給額の推移、支給vs控除の内訳をグラフ表示
- **スマホ対応**: モバイルファーストなUI（ボトムナビゲーション）

## 使い方

1. アップロードタブからPDFをタップして選択
2. 自動解析された内容を確認・修正
3. 保存するとダッシュボードに反映される

対応PDFフォーマット:
- GMOリサーチ&AI株式会社の給与明細（支給明細書）
- 給与所得の源泉徴収票

## 画面構成

| ページ | 内容 |
|---|---|
| ダッシュボード | 最新月のサマリ・月次推移グラフ |
| 給与明細 | 月次一覧・詳細（支給/控除/勤怠） |
| 源泉徴収票 | 年次データ・給与明細からの年次集計 |
| アップロード | PDFアップロード・自動解析・登録 |

## データの保存について

すべてのデータはブラウザの `localStorage` に保存されます。

- ブラウザのデータを削除するとデータも消えます
- 異なるブラウザ・デバイス間でデータは共有されません

## 開発環境のセットアップ

```bash
npm install
npm run dev
```

ローカルで http://localhost:5173/payslip-tracker/ にアクセス。

```bash
npm run build    # 本番ビルド
```

## 技術スタック

- **フレームワーク**: React 18 + Vite + TypeScript
- **スタイル**: Tailwind CSS v3
- **グラフ**: Recharts v2
- **PDF解析**: pdfjs-dist v5（ブラウザ内処理）
- **状態管理**: Zustand v5
- **ストレージ**: localStorage
- **ルーティング**: React Router v6
- **デプロイ**: GitHub Pages（GitHub Actions）

## デプロイ

`main` ブランチへのプッシュで GitHub Actions が自動ビルド・デプロイします。
